- **Edge case: Codex CLI version too old to recognize new manifest fields** — not currently covered. **Decision:** acceptable risk, fail-soft behavior is the user's responsibility (run a recent Codex version). Documented in Assumption Audit below.

- **Error handling: Codex marketplace template missing** — same pattern: compile script skips with warning.

No critical gaps.

- **GitHub Action that auto-tags on version push**: watches `src/.claude-plugin/plugin.json` (per CHANGELOG history). Unaffected by output-path changes. **Verification:** check `.github/workflows/` after compile to confirm.
